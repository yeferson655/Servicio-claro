# 📱 Gestión de Servicios - Claro Colombia
Este proyecto es una interfaz web diseñada para facilitar la administración y el registro de usuarios en los servicios de Claro.

### 🚀 Funcionalidades
* Formulario de captura y validación de datos.
* Interfaz optimizada para una navegación fluida en celulares.
* Diseño visual basado en la identidad oficial de la marca.
